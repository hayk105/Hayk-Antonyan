Git-ի աշխատանքային հոսքը բաժանված է երեք վիճակի.

1. Աշխատանքային գրացուցակ - Փոփոխեք ֆայլերը ձեր աշխատանքային գրացուցակում
2. Բեմականացման տարածք (Ինդեքս) - Բեմադրեք ֆայլերը և ավելացրեք դրանց նկարները ձեր բեմադրության տարածքում
3. Git գրացուցակ (պահեստ) - Կատարեք մի հանձնում, որը մշտապես պահում է լուսանկարները ձեր Git գրացուցակում: Ստուգեք առկա ցանկացած տարբերակ, կատարեք փոփոխություններ, բեմադրեք դրանք և կատարեք:
